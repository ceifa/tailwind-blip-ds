# Blip Design System Plugin for Tailwind CSS

## Installation

Install the plugin from npm:

```bash
npm install -D tailwind-blip-ds
```

Then add the plugin to your `tailwind.config.js` file:

```js
// tailwind.config.js

module.exports = {
  theme: {
    // ...
  },
  plugins: [
    require('tailwind-blip-ds'),
    // ...
  ],
}
```
